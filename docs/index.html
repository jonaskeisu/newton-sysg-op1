Kurslitteratur

Kursmaterialet består av tillhandahållna anteckningar från föreläsningarna, övningarna och inlämningsuppgifterna samt nedanstående kurslitteratur:

C# 8.0 and .NET Core 3.0 – Modern Cross-Platform Development, 4th Edition av Mark J. Price

Fördjupningsmaterial

Följande resurser innehåller rekommenderat fördjupande material inom områden som täcks i kursen:
  • Microsoft Docs - omfattande dokumentation om allt gällande Microsofts produkter, verktyg och teknologier, bl.a. C# / .NET Core och Visual Studio.
  • Pro Git - välskriven gratis bok som går på djupet med Git och beskriver syftet med GitHub.
  • Stack Overflow - mycket populärt diskussionsforum för mjukvaruutvecklare, där erfarna utvecklare svarar på frågor gällande utveckling med alla populära programmeringsspråk.

Utvecklingsverktyg

Utvecklingsverktyget för kursen är Visual Studio Code med inbyggt terminalfönster.

För versionhantering av källkod används Git.

För delning av versionshanterad källkod används webbtjänsten GitHub.

Kommunikationsverktyg

Slack workspace

För kommunikation med Slack används workspace: newton-sysg4.slack.com

Elever kan skicka direktmeddelanden till varandra och till examinatorn via ovanstående workspace.

Chatkanalen general är till för fri kommunikation mellan elever gällnde vad som helst som rör kursen, t.ex. ställa frågor öppna frågor eller dela med sig av tips.
